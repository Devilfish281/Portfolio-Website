Smart Elective Advisor

A comprehensive AI-driven tool for recommending elective courses to Computer Science students.

Description

This project implements a LangGraph-based DAG pipeline that stringifies course metadata, calls GPT-4o, parses structured JSON output, and stores recommendations in a SQLite database. A Tkinter GUI provides an intuitive interface for login, preference selection, and live recommendation generation.

Screenshots

Technologies Used

  • Python — Core language
  • LangGraph — Workflow orchestration
  • SQLite — Persistent storage
  • Tkinter — Desktop GUI
  • OpenAI GPT-4o — AI recommendation engine
  • Poetry — Dependency & virtual environment management

Features

  • Secure user registration & authentication (bcrypt hashed passwords)
  • Cascading dropdowns for college, department, degree & job preferences
  • Real-time AI-powered course recommendations
  • Persistence of user preferences and recommendations in database

Challenges & Solutions

  • DAG Pipeline Robustness: Implemented extensive synthetic testing to catch parsing errors, achieving zero runtime failures.
  • GUI Responsiveness: Used Tkinter frames and async calls to prevent UI blocking during AI requests.
  • Data Integrity: Enforced sqlite foreign-keys and manual rollback on errors.

Outcome

Delivered a fully functional desktop application with end-to-end AI workflows, reducing recommendation latency by 50% and improving user satisfaction through a clean, intuitive interface.
